# Tableau
Análisis de audiencias, engagement y comportamiento del perfil de IG de la empresa enGira!
📊 Análisis del rendimiento de una cuenta de Instagram
Audiencia, engagement y comportamiento mediante dashboards interactivos
📌 Descripción del proyecto
Este proyecto analiza el rendimiento de una cuenta de Instagram mediante el uso de técnicas de análisis de datos y visualización interactiva en Tableau.
A partir de los datos obtenidos mediante la Instagram Graph API, se estudian tres dimensiones principales del comportamiento de la audiencia:
Engagement con el contenido
Perfil demográfico y geográfico de la audiencia
Patrones de interacción con el perfil
El objetivo del análisis es comprender cómo interactúan los usuarios con el perfil y qué factores influyen en el rendimiento del contenido, con el fin de identificar oportunidades de optimización en la estrategia de comunicación digital.
🗂️ Estructura del proyecto
El proyecto se organiza en tres dashboards principales que analizan diferentes aspectos del comportamiento de la audiencia.
1️⃣ Dashboard de Engagement
Analiza el nivel de interacción que generan las publicaciones.
Indicadores principales:
Engagement rate
Número de interacciones
Alcance de las publicaciones
Engagement por tipo de contenido
Principales insights:
El perfil presenta un engagement del 7,55 %, indicando una audiencia activa.
Los carruseles generan mayor interacción que imágenes o vídeos.
El engagement muestra variaciones a lo largo del tiempo, con picos en determinados periodos.
2️⃣ Dashboard de Audiencia
Describe el perfil demográfico y geográfico de los seguidores.
Variables analizadas:
Distribución por género
Distribución por grupos de edad
Distribución geográfica por países y regiones
Principales insights:
La audiencia está ligeramente feminizada (47,8 %).
El grupo predominante es 35–44 años.
La audiencia se concentra principalmente en España, con un peso destacado de Madrid (53,2 %).
3️⃣ Dashboard de Comportamiento
Analiza cómo interactúan los usuarios con el perfil.
Métricas analizadas:
Visitas al perfil
Cuentas que interactúan
Interacciones totales
Clics al sitio web
Tipos de interacción (likes, comentarios, compartidos, guardados)
Principales insights:
El perfil genera 2213 visitas al perfil y 214 cuentas que interactúan.
Se registran 543 clics al sitio web, lo que indica capacidad de conversión.
La mayoría de las interacciones corresponden a likes, mientras que comentarios o compartidos son menos frecuentes.
🛠️ Herramientas utilizadas
Python
extracción y procesamiento de datos mediante la Instagram Graph API
limpieza y transformación de datasets con Pandas
Tableau
creación de dashboards interactivos
análisis visual de métricas de rendimiento
📊 Fuentes de datos
Los datos utilizados proceden de la Instagram Graph API, que permite acceder a métricas de rendimiento del perfil y de las publicaciones.
Entre las métricas analizadas se incluyen:
reach
impressions
accounts engaged
profile views
website clicks
likes
comments
shares
saves
🎯 Objetivo del análisis
El objetivo principal del proyecto es extraer insights accionables sobre el comportamiento de la audiencia, que permitan optimizar la estrategia de contenido en redes sociales.
El análisis permite identificar:
qué tipos de contenido generan mayor interacción
qué perfil demográfico compone la audiencia
cómo interactúan los usuarios con el perfil
en qué medida el perfil genera tráfico hacia otras plataformas
📈 Conclusiones
El perfil presenta niveles de engagement relativamente elevados, lo que indica que el contenido resulta relevante para la audiencia.
El formato de publicación influye significativamente en el rendimiento, destacando los carruseles como el tipo de contenido con mayor capacidad de interacción.
La audiencia se concentra principalmente en España y en usuarios adultos entre 25 y 44 años, lo que sugiere un perfil demográfico relativamente definido.
Por último, el perfil no solo genera interacción dentro de la plataforma, sino que también actúa como canal de tráfico hacia otros contenidos, evidenciado por el número de clics al sitio web.
